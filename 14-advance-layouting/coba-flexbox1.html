Mengenal Flexbox Container

1. display: flex

Secara default (Normal Flow), elemen div menumpuk ke bawah. Begitu kita beri display: flex, mereka langsung berjejer ke samping.

2. flex-direction (Menentukan Sumbu/arah)

Menentukan arah barisan (Main Axis). Ingat: Jika direction berubah, maka fungsi alignment/perataan juga ikut berubah arah!

3. justify-content (Alignment di Main Axis)

Mengatur bagaimana item berbaris di Main Axis. Sangat berguna untuk mendistribusikan ruang kosong.

2
3

4. gap

Dulu kita butuh margin untuk memisahkan kotak. Sekarang cukup gunakan gap di parent-nya. Kelebihannya: tidak ada margin sisa di item terakhir!

5. align-items (Alignment di Cross Axis)

Mengatur perataan/alignment di Cross Axis (tegak lurus dengan arah barisan). Di sini wadah diberi tinggi 300px agar terlihat bedanya.

6. flex-wrap & align-content

flex-wrap membuat item pindah ke baris baru. Jika sudah ada multi-baris, kita gunakan align-content untuk mengatur jarak antar tumpukan baris tersebut.
